# Personal Web Portfolio

The website includes a clean layout with sections like Home, About Me, Projects, and Contact.

## Features
- Fully responsive design
- About Me section with introduction
- Projects section to highlight my work
- Contact form / contact details
- Simple and user-friendly UI

## Technologies Used

- CSS
- JavaScript

## Purpose
This portfolio is made to present my work in a professional way and create an online presence for job opportunities, university applications, or freelancing.
